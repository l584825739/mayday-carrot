# 五月天卜卜音乐互动网页

一个以五月天卜卜为主题的互动式音乐网页，通过歌词暗号解锁宝箱，探索五月天的音乐时光机！

## 🌟 功能特点

- **歌词暗号解锁**：通过输入正确的歌词解锁神秘宝箱
- **专辑时光机**：按时间顺序展示五月天专辑，滚动到最前端的专辑会自动播放对应的经典歌曲
- **响应式设计**：完美适配手机移动端和桌面端
- **温馨五月天主题**：使用温暖的橙色系配色，符合五月天卜卜的可爱形象

## 🚀 如何使用

1. 打开网页，看到第一句歌词提示
2. 输入正确的下一句歌词（密码）
3. 解锁宝箱后，点击阿信卜卜图片进入专辑展示页面
4. 滚动浏览专辑，自动播放对应歌曲
5. 使用底部播放器控制播放/暂停

## 🎨 设计理念

- **温暖色彩**：使用橙色和米色为主的温暖色调，呼应五月天卜卜的形象
- **流畅动画**：添加适度的过渡动画，提升用户体验
- **直观交互**：简单明了的用户界面，易于操作
- **音乐沉浸**：通过专辑封面和音乐播放，营造沉浸式音乐体验
